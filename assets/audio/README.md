# Audio Assets

This directory contains audio assets for the Unseen app.

## Required Audio Files

### Ambient Sounds
- `ambient_drone.mp3` - Background ambient horror drone (loopable, 2-3 minutes)
- `whispers.mp3` - Whispering sounds (loopable, 1-2 minutes)

### Sound Effects
- `jump_scare.mp3` - Jump scare sound effect (short, 1-2 seconds)
- `heartbeat.mp3` - Heartbeat sound (loopable, 30 seconds)
- `footsteps.mp3` - Footsteps sound effect (loopable, 30 seconds)
- `clue_found.mp3` - Sound played when clue is found (2-3 seconds)

## Audio Specifications
- Format: MP3
- Sample Rate: 44.1kHz or 48kHz
- Bitrate: 128-192 kbps
- Horror theme: Dark, eerie, unsettling

## Notes
- Loopable files should be seamless loops
- Keep file sizes reasonable (< 5MB per file)
- For now, the app handles missing audio gracefully (no crashes)
