# Image Assets

This directory contains image assets for the Unseen app.

## Required Images

### Main Images
- `logo.png` - App logo (recommended: 512x512px, transparent background)
- `splash_bg.png` - Splash screen background (recommended: 1080x1920px, dark horror theme)

### Stickers (in `stickers/` subdirectory)
Placeholder sticker images for photo mode:
- `eye.png` - Eye sticker
- `warning.png` - Warning sticker
- `blur.png` - Blur effect sticker
- `fire.png` - Fire sticker
- `bug.png` - Bug sticker
- `mind.png` - Mind/psychology sticker
- `moon.png` - Moon sticker
- `flash.png` - Flash sticker

## Notes
- All images should follow the horror theme (dark, blood red accents)
- Use PNG format with transparency where needed
- For now, the app uses icon placeholders if images are missing
